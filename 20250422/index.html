hml 5

HTML5 is the fifth version of HTML (HyperText Markup Language) — the standard language used to create and structure content on the web. It was developed by the W3C and WHATWG to modernize the web and make it more powerful and flexible.

css

CSS, which stands for Cascading Style Sheets, is a language used to describe the presentation and visual design of a webpage. While HTML structures the content, CSS determines how that content looks—such as colors, fonts, margins, alignment, and overall layout

javascript

JavaScript is a powerful, versatile programming language commonly used to add interactivity, dynamic features, and functionality to websites. While HTML structures the content and CSS styles it, JavaScript brings it to life by enabling things like animations, form validations, interactive maps, and more
